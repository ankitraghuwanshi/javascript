  1. Click on '+' button to add ticket/task ,then choose color based on ticket/task state, then type ticket/task text, then press shift key
  2. Click on 'x' button to remove ticket/task,after pressing 'x' button click on 'ok' button ,it means now you can remove/delete ticket/task, click on ticket/task that you want to remove/delete
  3. Note:-you can delete as many ticket/task as you want
  4. If you click 'x' button again ,then 'ok' button ,now you cann't remove/delete any ticket/task
  5. If you want to change the text/content of ticket/task then, you should first click on that ticket/task lock icon,by doing that lock will open, now you can change the text/content, if you press again the open-lock icon, now you cann't change the text/content of ticket/task
  6. On top left, if you click on any color then ticket/task gets filtered and you can now only sees ticket/task having color that you press
  7. Means from filtering, you can focus on how many ticket/task are there in that state
  8. Remember: pink=backlog, yellow=pending, green=overview, blue=done
  9. you can change the color/state of particular ticket/task by clicking on that ticket/task color
